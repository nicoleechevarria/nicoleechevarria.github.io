Website Evaluations

Duolingo Website

Duolingo is a personalized language learning service geared toward giving consumers small, managable assignments so that they can learn a completely different langauge.

  • Contrast: Duolingo has a beautiful theme where blue backgrounds have white text and white backgrounds have black text.
  • Repitition: Everything from the font to the graphics of their mascot, Duo, are consistent throughout the whole website.
  • Allignment: Every section is centered and all the text align to the left
  • Proximity: From the text boxes to the sections themselves, either the background changes color or there is a divider.

Exmouth View Website

This website is a means of getting anyone viewing it to learn some of the history of the Exmouth View Hotel.

  • Contrast: There is clear contrast between the header, footer and body and even the side panel.
  • Repitition: The only source of repetition is the text which is all in a uniform font.
  • Allignment: Every section has the same margin and padding except for the footer. Thats the only reason it seems unaligned at the bottom.
  • Proximity: The header could be thinner so that it does not seem like the only thing on the page when its first open. Other than that there is even spacing and such.
